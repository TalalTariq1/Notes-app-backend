# 📝 Notes App Backend (Express + EJS + MongoDB)

A simple CRUD-based Notes Application built using Node.js, Express, MongoDB, and EJS.  
This project focuses on backend development fundamentals, database integration, and server-side rendering.

---

## 🚀 Features

- Create notes  
- View all notes  
- Update notes  
- Delete notes  
- Server-side rendering using EJS  
- MongoDB database integration  

---

## 🛠️ Tech Stack

- Node.js  
- Express.js  
- MongoDB (Mongoose)  
- EJS  
- HTML & CSS  

---

## 📌 Project Overview

This project demonstrates a basic full-stack backend application where notes are managed through CRUD operations and rendered dynamically using EJS templates.
